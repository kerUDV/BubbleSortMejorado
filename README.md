# Implementación de Bubble Sort Mejorado en Java

## Descripción

Este proyecto implementa el algoritmo Bubble Sort tradicional y una versión optimizada que incluye:

- Detección temprana mediante variable booleana (`swapped`)
- Uso del índice del último intercambio (`lastSwap`) para reducir el rango de comparación

Se realizaron pruebas en diferentes escenarios:

- Arreglo ordenado
- Arreglo en orden inverso
- Arreglo aleatorio
- Arreglo casi ordenado

Se midieron:
- Número de comparaciones
- Número de intercambios
- Tiempo de ejecución en nanosegundos

Los resultados fueron exportados a un archivo CSV y posteriormente graficados en PDF.
